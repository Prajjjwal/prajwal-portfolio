# prajwalkakade.dev

Freelance web developer portfolio — Next.js 14 (App Router) + Tailwind CSS + Framer Motion.

## Develop

```bash
npm install
npm run dev
```

## Deploy (Vercel)

Push to a Git repo and import it at vercel.com — zero config needed. Set the
custom domain `prajwalkakade.dev` in the Vercel dashboard.

## Editing content

All site content (projects, testimonials, stats, contact links, profile URLs)
lives in `lib/data.ts`. Swap the placeholder LinkedIn/Upwork/Fiverr URLs there
when profiles are ready.

The downloadable resume PDF is `public/Prajwal-Kakade-Resume.pdf` — replace the
file to update it. The `/resume` page is also print-optimized ("Print / Save
PDF" button).
